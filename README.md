The contents of this project have analysis,data and visual insights on sales of a Bright Coffe Shop for a CEO.
My key insight while i was working on the project i gathered from the data is that we can see the product that is making most sales is Coffee and the least product to make sales is Packed Chocolate,This give more information on if we should promote Packaged Chocolate or stop stocking it altogether and increase Coffee stock.
